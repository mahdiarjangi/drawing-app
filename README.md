
# Drawing App

Drawing app with pure javascript


## Installation

Install my-project with npm

```bash
  cd drawing-app
  npm install
```
    
## Deployment

To deploy this project run

```bash
  npm run dev
```


## Technologies

HTML (canvas), CSS, JavaScript


## Features

- Undo Last Change
- Change Pen Width
- Create Image From Canvas
